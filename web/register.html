On a phone and detection didn't work? Enter the MAC manually

Find your device's Wi-Fi MAC address: iOS — Settings → Wi-Fi → (i) next to the network → "Wireless Address"/"Private Wi-Fi Address". Android — Settings → About phone → Status → Wi-Fi MAC address. Windows — run getmac. Mac/Linux — run ifconfig or ip link.
